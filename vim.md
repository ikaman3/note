# Vim
vim 명령어 및 단축키를 정리하는 마크다운 문서

# String(문자열)  

## /string  
문자열을 검색하여 해당 위치로 이동  
검색한 상태에서  
- n : 아래에서 문자열 찾기
- N (Shift + n) : 위에서 문자열 찾기 
```
/cover_1.jpg
```

## %s/source/target
특정 문자열을 찾고, 원하는 문자열로 대체
- % : 파일 전체가 대체 명령어 실행 범위
- source : 변경하려는 원본 문자열
- target : 대체할 문자열
- c : 문자열을 대체할 때마다 확인
    - y : 대체
    - n : 건너뛰기
    - a : 모두 대체
```
%s/abc/def

%s/abc/def/c
```
